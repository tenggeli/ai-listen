# listen 接口说明

## 1. 约定
- 前缀：`/api/v1`
- 后台前缀：`/api/v1/admin`
- 鉴权：`Authorization: Bearer <token>`
- 返回结构：`code`、`message`、`data`、`traceId`
- 后台鉴权模型：管理员登录 + 路由粒度权限点（RBAC）

## 2. 当前已接通核心接口
