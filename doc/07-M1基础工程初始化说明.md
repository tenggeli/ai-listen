  - `provider-center`

  - `complaint`

### 3.2 前端

## 4. 存储与运行说明
- 默认情况下，后端使用内存存储，适合快速启动和本地测试
- 配置环境变量 `MYSQL_DSN` 后，应用会启用 MySQL 存储
- MySQL 模式下，应用启动时会自动执行初始化建表 SQL，并初始化基础服务项目数据
- 当前 MySQL 已覆盖：
  - 用户认证
  - 服务提供方入驻与审核
  - 订单状态机与订单主链路
