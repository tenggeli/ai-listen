## 7. 推荐派单方式

优先派发：

1. 一个原型页 + 对应 API Adapter
2. 一个后端上下文模块 + migration + 单测
3. 一个状态机（订单/投诉/审核）
4. 一个配置与仓储对齐任务

不建议派发：

- “一次做完整产品”
- “跨 3 个端的大而全任务”
- “先做真实第三方对接”
